A WordPress plugin for creating online polls with featured images, social sharing, and vote tracking.

== Description ==
The **Online Voting and Poll** plugin provides an easy-to-use solution for creating and managing polls and voting sessions directly on your WordPress site. It includes:
- Featured images for polls.
- Vote tracking.
- Social sharing buttons (Facebook, Twitter, LinkedIn, etc.).
- Responsive design.
- Admin panel for tracking and visualizing results.

This plugin is ideal for websites that want to engage visitors through surveys or voting contests.

== Features ==
* Create unlimited polls and voting sessions.
* Add featured images and descriptions for each poll.
* Users can vote on polls directly from the page.
* AJAX-based voting to prevent page reload.
* Social sharing buttons for each poll (Facebook, Twitter, WhatsApp, etc.).
* Vote tracking and visual analytics in the admin area.
* Supports Bootstrap and Font Awesome.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/online-voting-and-poll` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the **Polls** menu item in the WordPress admin area to create and manage polls.

== Frequently Asked Questions ==

= Can I customize the poll appearance? =
Yes, the plugin provides several customization options, including typography (using Redux) to change the font family for poll titles.

= Can users share their votes on social media? =
Yes, social media sharing options are available to make it easy for users to share polls on platforms like Facebook, Twitter, and WhatsApp.

= Is there a limit to the number of polls I can create? =
No, you can create as many polls as you'd like.
